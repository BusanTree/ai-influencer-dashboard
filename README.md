# AI 인플루언서 관리 대시보드

AI 인플루언서 프로필과 이미지/영상 자료를 한 화면에서 관리할 수 있는 웹 대시보드입니다.

## 주요 기능

- 인플루언서 목록 확인과 검색
- 인플루언서, 미디어 보관함, 캠페인 일정 화면 전환
- 상태별 필터
- 프로필 정보 편집
- 인플루언서별 이미지와 영상 첨부, 미리보기, 삭제
- 캠페인 일정표와 승인 대기 작업 확인
- 라이트 모드와 다크 모드 전환
- 브라우저 안에 자료 저장

## 사용 방법

`index.html` 파일을 브라우저에서 열면 바로 사용할 수 있습니다.

GitHub Pages로 배포하면 별도 설치 없이 웹 주소로 접속할 수 있습니다.
